# TFG: Predicció de la Vulnerabilitat Energètica a Barcelona

**Alumne:** Jordi  
**Grau:** Enginyeria Industrial (ETSEIB)

## Descripció del Projecte
Aquest repositori conté el desenvolupament d'un sistema de Deep Learning per predir l'Índex de Vulnerabilitat (IDV) a nivell de secció censal a Barcelona, basat en tres pilars: constructiu, climàtic i socioeconòmic.

## Estructura del Repositori
* `notebooks/`: Codis d'entrenament (Regressió, MLP, LSTM).
* `data/`: Dades processades i logs de resultats.
* `models/`: Models entrenats (en fases posteriors).
